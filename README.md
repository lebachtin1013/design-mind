# Design Mind Newsletter

Welcome to the Design Mind Newsletter repository!

This is the home of Techcombank's Design Mind newsletter, showcasing design excellence and insights from the Design Centre of Excellence (DCoE).

## Contents

- `index.html` - Main newsletter page
- `assets/` - Images and media files for the newsletter

## Deployment

This project is ready to deploy on Vercel:

1. Go to [vercel.com](https://vercel.com)
2. Click "Add New Project"
3. Import this repository
4. Click "Deploy"

Your site will be live at `https://design-mind.vercel.app`

## About

Design Mind invites you to peek behind the curtain and discover how user experience and interfaces are being crafted at Techcombank.
